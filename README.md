# DEKIRU WP Theme 2025

Updated to use @use instead of @import\
Last tested with Node v25

Install packages\
`pnpm install` 

Dev\
`pnpm start`

Production (if not using autoptimize)\
`pnpm build`


__Required plugins__
* ACF Pro/SCF
* Classic Editor +
* Safe SVG

__Recommended plugins__
* ACF Autosoze
* Crop Thumbnails
* Disable Comments
* PNG to JPG
* Regenerate Thumbnails
